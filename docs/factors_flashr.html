Introduction

In this part, we used ‘flashr’, to extract factors from gene expression matrix. We also computed the heritability of each factors using GCTA.

Material and Methods

  1. We identified 14 modules in wgcna_module part. We can obtain the gene expression data from GTEx dataset for each module and got 14 gene expression matrices.

  2. We used workflow showed in ‘flashr’ to extract factors.

f = flash(Y)    ### Y is the gene expression matrix.
ldf = flash_get_ldf(f)
  1. Then, we used the backfitting algorithm to improve the fit (but it takes longer).
f.b = flash(Y, f_init = f, backfit=TRUE, greedy=FALSE)
ldf.b = flash_get_ldf(f.b)
  1. Computed the heritability of each module.
